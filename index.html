Partie 1

Comment fonctionne un site web

Un site web repose sur trois éléments principaux qui fonctionnent ensemble pour afficher une page dans votre navigateur.

Structure
HTML
L'ossature de la page, comme les murs d'une maison.
Apparence
CSS
Les couleurs, la mise en page, les polices.
Interactions
JS
Les animations, boutons, menus dynamiques.

Quand on tape une adresse dans un navigateur, celui-ci envoie une demande à un serveur, qui renvoie les fichiers du site. Le navigateur les lit, assemble le HTML, applique le CSS, exécute le JavaScript — et affiche la page.

Navigateur → Requête serveur → Fichiers reçus → Page affichée
Partie 2

Les métiers du web

Le domaine du web m'attire parce qu'il combine créativité, logique et technologie. On peut créer quelque chose à partir de rien, résoudre des problèmes, et voir immédiatement le résultat.

Les métiers possibles
Développeur web
Écrit le code, crée les fonctionnalités, assure le bon fonctionnement.
Webmaster
Gère un site au quotidien, met à jour les pages, corrige les bugs.
Développeur front-end
S'occupe de la partie visible, ce que l'utilisateur voit.
Développeur back-end
Gère la partie cachée : bases de données, serveurs.
Webdesigner
Crée l'apparence du site : couleurs, images, mise en page.
Admin système
S'occupe des serveurs et de l'hébergement du site.
Compétences nécessaires
Savoir coder (HTML, CSS, JavaScript, Python, PHP…)
Avoir de la logique pour résoudre des problèmes
Être créatif pour imaginer des interfaces
Savoir travailler en équipe — les projets se font rarement seul
Être curieux, car les technologies évoluent très vite

Conclusion

Comprendre comment fonctionne un site web permet de découvrir un univers beaucoup plus vaste : celui des métiers du numérique.

Ce thème m'a permis de mieux connaître les technologies du web, mais surtout d'explorer un parcours professionnel qui m'intéresse vraiment.

Les métiers du web sont variés, accessibles, et surtout tournés vers l'avenir. C'est un domaine où l'on peut créer, apprendre, et évoluer toute sa vie.
